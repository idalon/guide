# Beginner Builds

A new HUNTER isn’t necessarily a new player, so we assume you already have most of the mods. If you’re truly just starting out, you can adjust a few things if needed.

### Weapons

We’re providing a rivenless build, but if you do have a Riven, we recommend checking the [“Riven Guide”arrow-up-right](https://docs.google.com/document/d/17GKu52mvzQVJ5mcfDTcKP1q-mH6aMywsF3aNERU0E1w/edit?tab=t.0#heading=h.r7pdkh7485g1) to see the suggested stats.

### Companions

Sentinel-type companions are the most suitable. “Beast” or “Robotic Beast” companions are not recommended, as they will mostly just get in your way.
