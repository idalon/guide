# Intrinsics - Railjack & Duviri

All of this is optional, but if you're aiming for speed and efficiency, you should consider unlocking these intrinsics:

### Duviri Intrinsic

* Combat Rank 9, +25% Operator damage

Link: [hashtag](intrinsics-railjack-and-duviri.md#duviri-intrinsic)

### Railjack Intrinsics

* Tactical Rank 8, reduced Archwing blink cooldown
* Piloting Rank 8, +20% Archwing speed
* Gunnery Rank 8, +20% Archwing Strength, Range and Efficiency
* Command Rank 9, Unlock On-Call Crew

Link: [hashtag](intrinsics-railjack-and-duviri.md#railjack-intrinsics)
