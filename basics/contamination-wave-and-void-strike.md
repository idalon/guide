# Contamination Wave & Void Strike

When it comes to your Operator's abilities, it's important to understand how they work to maximize their effectiveness:

* **First Ability - Void Strike:** Provides a 10% damage increase for every percent of energy used, lasting 8 seconds with a 40 second cooldown. You can achieve up to a 1000% damage buff when your Operator's energy is at 100%.
* **Second Ability - Contamination Wave:** Launches a wave that debuffs enemies, making them 50% more vulnerable to Operator damage. This effect lasts 20 seconds and is not refreshable.

Void Strike (VS) is especially reserved for Water Limbs, as that's when the Eidolon's shields are tougher. However, with proper timing, you can potentially use Void Strike multiple times during each Eidolon fight. Contamination Wave (CW) is a bit more complicated to time properly, as you must wait for the first CW debuff to expire before you can apply it again.

{% hint style="warning" %}
Important: For fast limb break times, where the interval between limbs is close to 17.2 seconds, you need to be mindful when casting Contamination Wave. Otherwise, some limbs might not have the CW debuff active despite your best efforts.
{% endhint %}

You can manage this in several ways. For example, in a four-limb sequence:

{% stepper %}
{% step %}
### First Limb

Cast CW early, well before the shields become vulnerable.
{% endstep %}

{% step %}
### Second Limb

Cast CW late, as close as possible to when the shields become vulnerable.
{% endstep %}

{% step %}
### Third Limb

Skip CW and use VS instead.
{% endstep %}

{% step %}
### Fourth Limb

Repeat from step 1 after using VS.
{% endstep %}
{% endstepper %}
