# Volt Solo Method (Shraksun)

You don’t have to follow this exactly. QC, lure management and water preparation can be adjusted.

{% stepper %}
{% step %}
### Entering the Plains

* Enter the Plains
* QC in the [linkarrow-up-right](https://youtu.be/Xce4AZBNfXQ)
{% endstep %}

{% step %}
### Terry (Teralyst) – Fight

#### First Limb

* Do Terry’s first limb
* Immediately go pick up 2 lures

{% hint style="warning" %}
If you are too slow:

* Take 1 lure first
* Pick up the second one after the next limb
{% endhint %}

#### Limb References

* For limb timings and positions, refer to the [dedicated guidearrow-up-right](https://docs.google.com/document/d/17GKu52mvzQVJ5mcfDTcKP1q-mH6aMywsF3aNERU0E1w/edit?tab=t.0#heading=h.g6c6j6nbrazx)
{% endstep %}

{% step %}
### Terry Down – Setup

Before finishing Terry:

* Make sure your lures are set to Hold Position
* Place your [crewmatearrow-up-right](https://docs.google.com/document/d/17GKu52mvzQVJ5mcfDTcKP1q-mH6aMywsF3aNERU0E1w/edit?tab=t.0#heading=h.lj9a4zg2ptu5)
{% endstep %}

{% step %}
### Garry Water Preparation

After the Terry down:

* Take at least 3 lures.
* Place your water setup motes (see loc pin position: [loc pin positionarrow-up-right](https://docs.google.com/document/d/17GKu52mvzQVJ5mcfDTcKP1q-mH6aMywsF3aNERU0E1w/edit?tab=t.0#heading=h.cmwi817ei3xb))

Then:

* Perform the Terry capshot
* Return to the shrine
{% endstep %}
{% endstepper %}
