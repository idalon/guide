# Harry Nova Setup

* Nova may need to reposition the LONG portal, as Lures may teleport when Volt places his shields.
* Since Nova’s rework, portals no longer have a duration, meaning you don’t need to constantly replace them.\
  The LONG portal is the only sensitive one.

Long Unstuck Setup – Nova

After cast the portal toward your Long tower, follow these steps:

{% stepper %}
{% step %}
Leave your Warframe in front of the portal.
{% endstep %}

{% step %}
Go into Operator.
{% endstep %}

{% step %}
Jump + sling 2–3 times toward the shrine, staying over water.
{% endstep %}

{% step %}
Swap back to Warframe + Archwing.
{% endstep %}

{% step %}
Never fly over land or the setup breaks.
{% endstep %}

{% step %}
Land near the tip rock of the Shrine.
{% endstep %}

{% step %}
Cast Roar for VOLT.
{% endstep %}

{% step %}
Keep /unstuck pre-typed in chat.
{% endstep %}
{% endstepper %}

At the Shrine – VS + Propa Timing

When the host detonates the loot:

* Go Operator
* Use the Power Transfer timer to time your VS:
  * 11s for Harry Water

![](../../../.gitbook/assets/image)

After the host inserts the shard:

Shoot the propas in this order (spam interact):

1. MID
2. CL
3. CR (just before or after your own shard inserts)

Then:

* /unstuck to Long
* Shoot the propa
* Turn around and break the limb
