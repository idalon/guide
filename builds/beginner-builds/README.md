# Beginner Builds

A new hunter isn’t necessarily a new player, so we assume you already have most of the mods.\
If you’re truly just starting out, you can adjust a few things if needed.

## Weapons

We’re providing a rivenless build, but if you do have a Riven,\
We recommend checking the “[<mark style="color:$success;">**Riven Guide**</mark>](https://app.gitbook.com/o/4Fp1ONKFMoRcF25FTbqR/s/SwotjVkQd1faB6N1f6j4/~/edit/~/changes/6/builds/riven-guide)” to see the suggested stats.

## Companions

Sentinel-type companions are the most suitable.\
“Beast” or “Robotic Beast” companions are not recommended, as they will mostly just get in your way.
