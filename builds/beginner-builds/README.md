# Beginner Builds

A new hunter isn’t necessarily a new player, so we assume you already have most of the mods.\
If you’re truly just starting out, you can adjust a few things if needed.

**Riven Mods**\
We’re providing a **rivenless builds** which work perfectly fine, but if you still want to use rivens then check the “[<mark style="color:$success;">**Riven Guide**</mark>](../riven-guide.md)”.
